# 隱私權政策

最後更新日期：2026-03-18

## 1. 蒐集的資料

本機器人僅為提供 Discord 伺服器管理與音樂功能，可能處理以下資料：
- Discord 使用者與伺服器基本識別資訊（例如：使用者 ID、伺服器 ID、頻道 ID）。
- 指令互動內容（例如：`/play`、`/settings` 參數）。
- 伺服器事件資料（例如：加入/離開、語音進出、訊息刪除與編輯事件）。
- 機器人運作所需的設定檔資料（`config.yml` 與 `guild-configs/*.yml`）。

## 2. 資料用途

資料僅用於：
- 執行機器人功能（播放音樂、通知、日誌、私人包廂等）。
- 產生伺服器管理日誌。
- 儲存伺服器個別設定。
- 除錯與維護服務穩定性。

## 3. 資料保存與安全

- 資料預設儲存在你部署機器人的主機本地。
- 你（伺服器管理者/部署者）需自行確保主機與檔案權限安全。
- 本專案不主動將資料上傳至作者自建伺服器（除 Discord API 與必要第三方服務請求外）。

## 4. 第三方服務

本機器人會與第三方服務互動以提供功能，例如：
- Discord API（指令、事件、訊息互動）。
- YouTube/Spotify 相關來源（僅用於音樂搜尋與播放）。

第三方服務之資料處理，依其官方政策為準。

## 5. 使用者權利

伺服器管理者可透過：
- 關閉對應模組（如通知/日誌）。
- 清理日誌訊息與設定檔。

以控制資料留存範圍。

## 6. 聯絡與責任

若你要公開提供此機器人，建議於公開頁面提供維護者聯絡方式，並明確告知伺服器成員資料處理方式。
